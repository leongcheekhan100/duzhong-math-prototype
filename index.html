Solve the following quadratic equation for x:
x² + 5x + 6 = 0
A
x = 2, x = 3
B
x = -2, x = -3
C
x = -1, x = -6
D
x = 1, x = 6
步骤 1 Step 1: 因式分解方程 Factor the equation
x² + 5x + 6 = (x + 2)(x + 3) = 0
步骤 2 Step 2: 令每个因子等于零 Set each factor to zero
x + 2 = 0 → x = -2
x + 3 = 0 → x = -3
答案 Answer: x = -2, x = -3 ✅
